## Project Description
CommunityHub is a full-stack community platform built progressively 
throughout IYF Weekend Academy Season 10. It allows users to create 
profiles, publish posts, interact through comments, and search for content.

## Technologies Used
### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js

## Features
- User profiles and authentication
- Posts and content creation
- Comments and interactions
- Search and filtering
- Responsive design
- Real-time features (stretch goal)

## How to Run
### Frontend
1. Clone this repository
2. Open `frontend/index.html` in your browser

### Backend
1. Navigate to the backend folder: `cd backend`
2. Install dependencies: `npm install`
3. Start the server: `npm start`
